# CreateNetFile
一个跨平台的pajek数据格式文件生成器

### 功能
* 高效迅速地将 <Tab> 分隔的文本文件转换成 pajek 程序所需的 .net 文件
* 词频统计功能

功能如下所示：
![功能图](./diagram.png)

### 支持平台
* MacOSX/Xcode
* Linux/g++
* Windows/Visual Studio
